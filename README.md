# Bank-Loan-Dashboard
The Bank Loan Dashboard is a data visualization project used to analyze and monitor loan-related data in a bank. It helps bank officials track loan applications, approvals, rejections, and pending cases in an easy and interactive way.

🔹 Features
✔ Total Loan Applications Count
✔ Approved / Rejected / Pending Loans
✔ Loan Amount Analysis
✔ Customer Category (Salary / Business)
✔ Region-wise Loan Distribution
✔ Monthly / Yearly Trends
✔ Interest Rate Analysis,
